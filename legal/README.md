# Circaevum Legal and Usage Docs

This folder centralizes policy and legal-operations documentation for Circaevum.

These documents are working drafts for business/legal review and are not legal advice.

## Core policy documents

- `ATTRIBUTION_POLICY.md`  
  Product attribution requirements (including overlay expectations).

- `TRADEMARK_POLICY.md`  
  Allowed and restricted use of the Circaevum name/logo and brand references.

- `TERMS_OF_USE_DRAFT.md`  
  Public-facing terms draft for users of hosted/web experiences.

## Commercial templates

- `COMMERCIAL_TERMS_DRAFT.md`  
  Base commercial terms draft.

- `ENTERPRISE_WHITE_LABEL_ADDENDUM_DRAFT.md`  
  Optional addendum draft for no-attribution/white-label rights.

## Licensing and compliance references

- `THIRD_PARTY_NOTICES.md`  
  Third-party software notices and attribution references.

- `LICENSING_STRATEGY_NOTES.md`  
  Open-core licensing decision notes and implementation checklist.

- `PROJECT_LICENSE_REFERENCE.md`  
  Pointer to the project license currently used by this web app.

## Vendor-level records (outside this folder)

Dependency-specific vendor records remain with vendored artifacts, for example:

- `../circaevum/js/vendor/astronomy-engine/VENDORED.md`
- `../circaevum/js/vendor/astronomy-engine/LICENSE.txt`
